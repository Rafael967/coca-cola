1 - O que é um repositório no Git e qual a sua importância?
São armazenados todos os arquivos e histórico de um projeto, ele é fundamental para o desenvolvimento de software, 
pois permite controlar as diferentes versões do projeto.

2 - Qual a função do Git Bash?
Fornecer uma interface de linha de comando para executar comandos git e gerenciar repositórios Git.

3 - O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
O comando git add é usado para adicionar arquivos. A diferença é que quando você executa o git add se adiciona um 
arquivo específico e o git add . você adiciona todos os arquivos.

4 - Por que utilizamos o ponto (.) no comando git add .?
É uma forma abreviada de dizer adicionar todos os arquivos no diretório atual.

5 - Explique o comando git commit e por que é importante criar uma mensagem descritiva.
O comando git comió é importante porque ele serve como um registro das alterações feitas no repertório.
Uma mensagem de commit é bom para entender o histórico do repertório e identificar alguma alteração.

6 - Qual a função do comando git push?
É usado para enviar as alterações  feitas no repertório.

7 - Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
É uma boa prática de desenvolvimento que ajuda a controlar as versões do projeto.

8 - Explique a função das tags HTML e como elas estruturam uma página.

9 - O que são atributos em uma tag HTML e qual a sua função?
